Prototype's Command Format

Format: [Namespace] [Action] [Argument]

Note: Argument is optional in some commands.

Namespaces:

  1. SOS - A SOS alarm to notify people about emergencies.
    Actions:
    • trigger: Trigger SOS
    • stop: Stop the alert system
  2. user_cmd - Command entered by the user (text or voice)
    Actions:
    • run: Run the entered command
  3. Appliance - Control a home appliance
    Actions:
    • light: Control the light
      Light Arguments:
      • on: Turn light on
      • off: Turn light off

    • fan: Control the fan
      Fan Arguments:
      • on: Turn fan on
      • off: Turn fan off
      • Number 0-4: Control fan speed
